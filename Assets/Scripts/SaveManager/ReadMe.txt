1.0.2版本变更日志。

* 商店列表更改
- 更新了资产的标题，去掉了资产名称后的 "(CG)" 字样。改为"|CG"，看起来更整齐。
- 更新了资产的主要描述来解释更多关于资产和它的功能。
- 更新了所有的链接，使资产的链接正确。
- 更新了关键图片，使其大小与主图相同。
- 更新了资产的配色方案到更柔和，旧的颜色有点突兀。

* 资产变更
+ 更新了资产编辑器窗口上的保存管理器标识。
+ 更新了卡特游戏导航标签菜单中保存管理器的优先级。
+ 纠正了导航标签菜单中保存数据编辑器的间距。
+ 更新了对所有代码的注释。
+ 更新了资产中的示例，以便用户可以引用它，而不需要以特定方式设置保存数据类。
+ 将所有示例代码命名为 CarterGames.Assets.SaveManager.Example，这样用户就不会意外地使用示例代码而不是资产本身。
+ 添加了一个新方法，以便在调用时重置保存文件。
+ 清空了 savedata 类，这样用户就可以马上添加自己的东西了。
