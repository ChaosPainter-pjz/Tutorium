Fantom Music Library

It is an original music material collection. It is collected as short as possible a seamless loop music, or repeatedly composed music.

In order to be able to use it for smartphone applications, I put in some versions of the file size reduced by adjusting the song length and bit rate. Please choose according to your application (There are only one type).

I have made a simple music player for the demo. It is good to build and play it on the actual machine. In addition, we use '.ogg' for the seamless loop. If you convert it to something not supported such as WebGL platform ('.aac', '.mp3' etc.), a little blank space will be inserted in the beginning (It is audio format specification).

Loop playback is possible even with something without "_Loop" (seamless loop) (There is a little at the end of the song).

------------------------------------------------

　オリジナル楽曲素材集です。なるべく短いシームレスなループ曲、または繰り返しで構成された曲を集めてあります。

　スマートフォンアプリなどにも使用できるように、曲の長さやビットレートを調整することによってファイルサイズを小さくしたバージョンをいくつか入れておきました。用途によってお好みで選んで下さい（１つしかないものもあります）。

　デモに簡単な音楽プレイヤーを作っておきました。適当にビルドして実機で再生してみるのも良いかもしれません。なお、シームレスループには '.ogg' を使用しています。WebGL など対応してないもの（'.aac' や '.mp3' 等）に変換した場合は、少し頭に空白が入ります（音声フォーマットの仕様です）。

　"_Loop"（シームレスループ）の付いてないものでも、ループ再生は可能です（曲の最後に少し間が入れてあります）。
