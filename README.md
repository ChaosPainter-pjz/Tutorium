# Tutorium
《全托教师》（又名《Tutorium》），这是一款基于Unity开发的游戏。改游戏已于2022年8月完全免费，于2023年2月正式开源。

这是我个人开发的一个游戏，所以游戏框架、程序结构等并没有什么参考价值，总而言之就是“依托答辩”。

但是，正是因为没有使用框架、抽象，代码的可读性很高，引用关系非常明确，代码复杂度低，注释较多。对于想开发类似游戏的初学者开是个不错参考。
